Wie funktioniert das Internet?

Webseiten bestehen aus Dateien, die von einem Webserver an einen Browser gesendet werden. Der Browser interpretiert HTML, CSS und JavaScript.

Die drei Grundtechnologien

  1. HTML – Struktur und Inhalt
  2. CSS – Aussehen und Layout
  3. JavaScript – Interaktivität

Werkzeuge

  • Browser (Chrome, Firefox)
  • Editor (z. B. VS Code)
  • Entwicklertools (F12)
